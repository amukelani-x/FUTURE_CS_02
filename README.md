# FUTURE_CS_02
Phishing Email Detection & Awareness System

## Project Overview
This report focuses on phishing email detection and cybersecurity awareness.

The aim of this report was to analyse email headers and identify phishing emails using free online cybersecurity tools.

Three email samples were analysed:
- One safe email
- Two phishing emails

# Objectives
The purpose of this project was to:

- Learn how phishing emails work
- Identify suspicious email signs
- Analyse email headers
- Understand SPF, DKIM, and DMARC
- Improve cybersecurity awareness

# Findings
The project found that phishing emails usually have:

- Fake or suspicious domains
- Failed SPF, DKIM, or DMARC checks
- Urgent messages
- Suspicious links
- Generic greetings
  
# Tools Used
- MXToolbox
Used to analyse email headers.

- URLScan.io
Used to inspect suspicious links safely.

- PhishTank
Used to check phishing URLs.
